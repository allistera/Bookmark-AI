Configure your Bookmark AI extension

The URL of your Bookmark AI API server (Cloudflare Worker)
How to use: 1. Make sure your Bookmark AI API is deployed (see the project README)
2. Enter your API endpoint URL above (default: https://bookmark-ai.allistera.workers.dev)
3. Click the extension icon when browsing to analyze and bookmark pages
4. Bookmarks will be automatically organized under "Bookmark AI" in your bookmarks bar
About Bookmark AI: This extension uses AI to automatically categorize and organize your bookmarks. It analyzes web pages and places them in the appropriate folder hierarchy.

For more information, visit the GitHub repository.
