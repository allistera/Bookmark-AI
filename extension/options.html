Configure your Bookmark AI extension

The URL of your Bookmark AI API server (Cloudflare Worker)
Your API key for authenticating with the worker. This should match the EXTENSION_API_KEY configured in your Cloudflare Worker.
Prepend a folder to all bookmark paths. For example, "Allisters Bookmarks" will save to "Allisters Bookmarks/Work_and_Engineering/..." instead of "Work_and_Engineering/...". Leave empty for no prepending.
How to get your API endpoint and key: 1. Deploy your Bookmark AI backend to Cloudflare Workers (see the project README)
2. Your API endpoint URL will be: https://bookmark-ai.[YOUR-ACCOUNT].workers.dev
3. Set the EXTENSION_API_KEY secret in your Cloudflare Worker or GitHub repository secrets
4. Enter the same API key in the "API Key" field above

To deploy:
• Run npm run deploy in the project directory, OR
• Push to the main branch to trigger automatic deployment via GitHub Actions

Security Note: Keep your API key secret! It authenticates your extension with the worker.
About Bookmark AI: This extension uses AI to automatically categorize and organize your bookmarks. It analyzes web pages and places them in the appropriate folder hierarchy.

For more information, visit the GitHub repository.
