Bookmark AI Settings

Configure your Bookmark AI extension

Your Anthropic API key for Claude AI. Get it from console.anthropic.com
Your Instapaper email address. Articles will be automatically saved to Instapaper if configured.
Your Instapaper account password.
Your Todoist API token. Get it from Todoist Settings → Integrations
How to get your API keys:

1. Claude API Key (Required):
• Visit console.anthropic.com
• Create an account or sign in
• Go to API Keys section and create a new key
• Copy the key starting with sk-ant-...

2. Instapaper (Optional - for auto-saving articles):
• Create an account at instapaper.com
• Use your email and password in the fields above

3. Todoist (Optional - for creating tasks):
• Go to Todoist Settings → Integrations
• Copy your API token from the Developer section

Security Note: Your API keys are stored using Chrome's sync storage, which synchronizes them across your devices via Google's servers. They are not sent to any third-party service except directly to the respective services (Claude, Instapaper, Todoist).
About Bookmark AI: This extension uses AI to automatically categorize and organize your bookmarks. It analyzes web pages using Claude AI and places them in the appropriate folder hierarchy. Articles are automatically saved to Instapaper if configured.
