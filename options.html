Security

API keys are stored in your browser's extension storage and synced across your Chrome devices. Use dedicated API keys with limited scope when possible.

How it works

When you save a bookmark, Claude AI analyzes the page content and automatically organizes it into the appropriate folder based on your existing bookmark structure.

Articles are sent to Instapaper. Other content is saved to Chrome bookmarks.
