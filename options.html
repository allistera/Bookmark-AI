Bookmark AI Settings

Configure your Bookmark AI extension

Claude AI Required
Your Anthropic API key for Claude AI. Get one at console.anthropic.com
Instapaper Optional
Articles detected by AI will be automatically saved to Instapaper if credentials are provided.
Leave blank if you don't have a password set on Instapaper.
Todoist Optional
Create tasks in Todoist when bookmarking (enable per-bookmark in popup).
Get your API token from Todoist Settings
Security Note: API keys are stored locally in your browser's extension storage and synced across your Chrome devices if sync is enabled. Use dedicated API keys with limited scope where possible.
How it works: 1. When you bookmark a page, Claude AI analyzes the content
2. Articles are automatically saved to Instapaper (if configured)
3. Other content is organized into bookmark folders based on AI categorization
4. Optionally create a Todoist task for follow-up

No backend server required! All processing happens directly from the extension.
